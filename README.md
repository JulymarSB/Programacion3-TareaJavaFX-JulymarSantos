## Aplicaciones realizadas

### 1. Registro de Visitantes

Interfaz para registrar visitantes de una institucion.

Incluye:

- Campos para nombre, cedula, telefono, motivo de visita, persona a visitar y fecha.
- ComboBox para seleccionar el tipo de visitante.
- RadioButton para seleccionar el documento entregado.
- Botones Registrar, Limpiar y Cancelar.
- TableView para mostrar los visitantes registrados.

---

### 2. Sistema de Inventario

Interfaz para registrar productos de un inventario.

Incluye:

- Campos para codigo, producto, categoria, precio, cantidad y proveedor.
- ComboBox para seleccionar el estado del producto.
- CheckBox para indicar caracteristicas del producto.
- Botones Guardar, Editar, Eliminar y Buscar.
- TableView para mostrar los productos registrados.

---

### 3. Perfil de Estudiante

Interfaz para visualizar y registrar la informacion de un estudiante.

Incluye:

- Foto simulada mediante un Rectangle.
- Campos para matricula, nombre, carrera, correo, telefono y direccion.
- ComboBox para seleccionar el semestre.
- ListView para mostrar las materias inscritas.
- TextArea para observaciones.
- Botones Guardar, Actualizar e Imprimir.

## Descripcion general

Este proyecto fue desarrollado utilizando JavaFX con el objetivo de practicar el diseno de interfaces graficas. Las tres aplicaciones fueron construidas unicamente con fines visuales, por lo que no incluyen eventos, logica de negocio, conexion a bases de datos ni almacenamiento de informacion.

El proyecto demuestra el uso de diferentes controles y layouts de JavaFX para crear interfaces organizadas, claras y faciles de utilizar.

---

## Observaciones

- No se implementaron eventos en los botones.
- No se utilizo conexion a base de datos.
- No se realizo almacenamiento en archivos.
- Todas las interfaces fueron desarrolladas siguiendo los requisitos establecidos para la tarea.
